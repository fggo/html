## Structure
```html
<html>
	<head> <!--info about page-->
		<title>web page title</title>
	</head>
	<body> <!--main browser window-->
		<h1>heading</h1>
		<p>paragraph</p>
		<h2>sub-heading</h2>
		<p>paragraph</p>
	</body>
</html>
```
## Text
```html
<p>
	<b>bold</b> text
	<i>italic</i> text
</p>

<p>
	E = MC<sup>2</sup> 
	H<sub>2</sub>O
</p>

<!--white space : two or more spaces collapse to one-->
<p>Hello World</p>
<p>Hello     World</p>

<!--Line Breaks-->
<p>Line One<br/>Line Two<br/>Line Three<br/></p>

<!--Horizontal Rules-->
<p>Current Topic</p>
<hr/>
<p>Next Topic</p>
```
